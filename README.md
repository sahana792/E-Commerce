\# E-Commerce Project



\## Project Overview

This is a Java-based E-Commerce web application developed using NetBeans IDE. The project simulates an online shopping system where users can register, log in, browse products, add items to cart, and place orders. It demonstrates full-stack development with frontend, backend, and database integration.



\## Features

\- User Registration and Login

\- Product Listing

\- Add to Cart functionality

\- Remove items from Cart

\- Order Placement system

\- MySQL Database connectivity

\- Session management for user login



\## Tech Stack

\- Java

\- JSP / Servlets

\- HTML, CSS, JavaScript

\- MySQL

\- NetBeans IDE

\- Apache Tomcat Server



\## How It Works

\- User registers and logs in

\- Products are fetched from database

\- User adds products to cart

\- Order is placed and stored in database
